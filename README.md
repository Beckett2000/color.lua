color.lua
===============

------ ------ ------ ------ ------

Supported Color Spaces:

------ ------ ------ 

```
RGB
HSV
HSB
HSL
HAI
HWB
HCG
HSM

CMY
CMYK

XYZ
LAB
LUV

LCHab
LCHub

TSL

YCbCr
YUV
YCgCo
YDbDr
```
